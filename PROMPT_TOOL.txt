Ferramenta de busca de produtos Dana Jalecos.

PARÂMETROS QUE VOCÊ PODE USAR:
• nome - palavra-chave do produto
• codigo - código do produto
• limit - quantidade de resultados (padrão: 10)

COMO PREENCHER:

Cliente: "tem jaleco?"
→ nome = "jaleco"
→ limit = 10

Cliente: "mostra gorro azul"
→ nome = "gorro azul"
→ limit = 10

Cliente: "busca código 300-MC-049"
→ codigo = "300-MC-049"
→ limit = 1

Cliente: "quero ver 5 aventais"
→ nome = "avental"
→ limit = 5

SEMPRE preencha pelo menos "nome" ou "codigo".
SEMPRE adicione "limit" (use 10 se cliente não especificar).

RESPOSTA DA API:
{
  "total": 279,
  "produtos": [
    {"nome": "Jaleco...", "preco": 79.90, "imagens": ["url"], "estoque": 150}
  ]
}

Interprete e responda naturalmente ao cliente.
