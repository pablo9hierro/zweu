IMPORTANTE: Você DEVE preencher os parâmetros abaixo quando usar esta ferramenta.

PARÂMETROS OBRIGATÓRIOS (preencha pelo menos UM):
• nome (string) - palavra-chave do produto
• codigo (string) - código do produto
• limit (number) - quantidade (padrão: 10)

INSTRUÇÕES CRÍTICAS:
1. NUNCA deixe os parâmetros vazios
2. SEMPRE extraia informações da mensagem do cliente
3. SEMPRE preencha "nome" quando cliente mencionar produto
4. SEMPRE adicione "limit" com valor numérico

EXEMPLOS OBRIGATÓRIOS:

Cliente: "tem jaleco?"
VOCÊ PREENCHE:
nome = "jaleco"
limit = 10

Cliente: "mostra gorro azul"
VOCÊ PREENCHE:
nome = "gorro azul"
limit = 10

Cliente: "código 300-MC-049"
VOCÊ PREENCHE:
codigo = "300-MC-049"
limit = 1

Cliente: "quero 5 aventais"
VOCÊ PREENCHE:
nome = "avental"
limit = 5

Se você enviar parâmetros VAZIOS, a requisição vai FALHAR.
SEMPRE preencha pelo menos "nome" com o produto que o cliente mencionou.
